// About

Security is a system,
not a checklist.

I've spent most of this decade in the intersection of networking, security governance, and infrastructure engineering. I declined a CISO offer to stay hands-on — because I believe the best security leaders understand the systems they're protecting at the deepest level.

Currently operating as a NetSec Engineer (tactical) and Cloud Security Architect (strategic) across multi-account AWS environments with hub-and-spoke topology, Transit Gateway, Check Point CloudGuard, and full Terraform IaC.

My approach: build security into the pipeline, not bolt it on after. From shift-left DevSecOps tooling to org-wide SCP enforcement, I design systems where doing the secure thing is the easy thing.
